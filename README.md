# Nex Drive

Sistema completo de gestao automotiva - estoque, clientes, financeiro e envio de mensagens em massa.

## Tecnologias

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Storage)

## Setup

```sh
git clone https://github.com/Hoolydev/nexdrive.git
cd nexdrive
npm install
npm run dev
```

## Deploy

Hospedado na Vercel com deploy automatico via GitHub.
